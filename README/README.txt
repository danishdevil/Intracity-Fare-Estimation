-------------------------------------------------------------------------------
answer.csv - submission file, contains the output generated by the script /notebooks_&_scripts/Driver.py

-------------------------------------------------------------------------------
data

This directory contains all the csv files which have data in them
	-processed_train.csv - preprocessed training file
	-processed_test.csv - preprocessed test file
	-intracity_fare_train.csv - original train file
	-intracity_fare_test.csv  -	original test file

-------------------------------------------------------------------------------
notebooks_&_scripts

This directory contains all the python scripts and jupyter notebooks associated with the problem
	- Driver.py : The main program file
					Takes the processed training and test files
					generates answer.csv
					Can also be used for cross validation testing
	- preprocess.py : This script preprocesses the intracity_fare_test and 
	intracity_fare_train files to generate processed_train.csv and processed_test.csv to be used by Driver.py

	-Driver.ipynb, preprocess.ipynb - corresponding jupyter notebooks

README
